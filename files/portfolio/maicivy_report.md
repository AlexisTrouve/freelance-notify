# Maicivy - Interactive CV Platform

## Tech Stack
- **Backend**: Go 1.24 (Fiber, GORM, PostgreSQL, Redis)
- **Frontend**: Next.js 14 (TypeScript, Tailwind, Framer)
- **AI**: Claude & OpenAI APIs
- **Infrastructure**: Docker, Prometheus/Grafana

## Metrics
**Size**: 30k+ LOC | **Backend**: 23.5k | **Frontend**: 8.9k | **Status**: Production-ready

## Purpose
Interactive CV with AI-powered motivation letters. Full-stack demo with JWT auth, WebSocket, activity tracking.

## Key Features
- Dynamic CV rendering
- AI letters (Claude/GPT-4)
- Real-time analytics & visitor tracking
- JWT authentication
- WebSocket updates
- Multi-language (i18n)
