# blockchain-core - Rust Blockchain Node

## Tech Stack
- **Language**: Rust 2021 (axum, tokio)
- **Crypto**: SHA-256, Ed25519 signatures
- **Serialization**: serde + JSON
- **Database**: JSON persistence

## Metrics
**Size**: 1.1k LOC | **Crates**: 2 | **Endpoints**: 11 | **Status**: MVP

## Purpose
Educational blockchain with PoW consensus, Ed25519 transactions, wallet management, and REST API. Account-based model with chain validation.

## Key Features
- PoW mining with block rewards
- Ed25519 transaction signatures
- Nonce-based replay protection
- Pending transaction pool
- Full chain validation
- JSON persistence
