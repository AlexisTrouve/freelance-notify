# GroveEngine - C++ Modular Game Engine

## Tech Stack
- **Language**: C++17, CMake
- **Renderer**: bgfx (2D/3D)
- **Audio/Input**: SDL2, spdlog

## Metrics
**Size**: 47.8k LOC | **Files**: 293 | **Tests**: 113 | **Status**: Development-ready

## Purpose
Experimental engine for rapid prototyping with 0.4ms hot-reload. 200-300 line modules with pub/sub IIO messaging.

## Key Features
- Hot-reload (0.4ms, state preservation)
- Pub/Sub messaging (IntraIO)
- 2D sprites, tilemaps, particles
- 10 UI widget types
- Sequential/Threaded modes
- Thread-safe design
