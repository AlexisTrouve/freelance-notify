# Confluent - Constructed Language Translation

## Tech Stack
- **Backend**: Node.js, Express.js (JWT auth)
- **LLM**: Claude & OpenAI GPT
- **Data**: JSON lexicon, Markdown docs

## Metrics
**Size**: 4k LOC | **Files**: 148 | **Docs**: 2.8k lines | **Status**: Production-ready

## Purpose
French ↔ Confluent translation engine for civjdr RPG. LLM-powered bidirectional translation with linguistic analysis.

## Key Features
- Bidirectional French ↔ Confluent
- 67+ radical system (agglutinative)
- Base-12 numbers, SOV syntax
- 2 language variants (Proto/Ancient)
- Web UI + REST API
- Admin panel (API keys, usage)
