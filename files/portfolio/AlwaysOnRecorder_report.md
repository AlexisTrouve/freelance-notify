# AlwaysOnRecorder - Android Audio Recording

## Tech Stack
- **Framework**: React Native 0.73 (TypeScript)
- **Audio**: react-native-audio-recorder-player
- **Database**: SQLite, AES-256 encryption
- **Background**: react-native-background-actions

## Metrics
**Size**: 5.2k LOC | **Services**: 6 | **Status**: MVP v0.1.0

## Purpose
Privacy-first continuous audio recording for Android with voice activity detection and local AES-256 encryption. No cloud auto-upload.

## Key Features
- Background recording (foreground service)
- Voice activity detection (30s+ silence)
- Local AES-256 encryption
- SQLite metadata & tagging
- Export/share multiple formats
- Android permissions management
